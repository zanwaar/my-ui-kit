Component

Buttons

Gunakan `.btn` untuk aksi. Variant dipakai untuk hierarki visual: primary untuk CTA utama, outline untuk aksi sekunder, ghost untuk aksi halus.

Preview
Usage

Class dasar yang wajib adalah .btn, lalu tambahkan variant seperti .btn-primary.

Guidelines
  • Gunakan `primary` untuk aksi utama per section.
  • Gunakan `outline` untuk secondary action.
  • Gunakan `danger` hanya untuk tindakan destruktif.
  • Jaga konsistensi ukuran tombol dalam satu area.
HTML
<button class="btn btn-primary">Save Changes</button>
<button class="btn btn-outline">Cancel</button>
<button class="btn btn-danger">Delete</button>
