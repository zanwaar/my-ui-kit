Tailwind UI Kit

Bangun UI cepat dengan class siap pakai.

My UI Kit adalah dokumentasi dan starter UI kit bergaya FlyonUI: punya docs, components, blocks, dan templates, tetapi memakai brand, class, dan aturan desain milik Anda sendiri.

Lihat Components Lihat Blocks
Install

Tambahkan file CSS dan JS dari folder core ke project Anda.

<link rel="stylesheet" href="core/dist/ui-kit.css">
<script src="core/js/theme-toggle.js"></script>

Components

Button, badge, alert, card, form, dan komponen kecil reusable.

Blocks

Kombinasi komponen siap pakai seperti hero, stats, pricing, dashboard.

Templates

Halaman utuh seperti admin dashboard, landing page, dan auth page.
