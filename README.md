# piceUI

A plug-and-play UI kit built with Tailwind CSS for rapidly building consistent user interfaces.

## Documentation

For full documentation, visit https://pice-ui.vercel.app

## Install

```bash
npm install piceui
```

## Usage

Import the main stylesheet in your global CSS:

```css
@import "piceui/dist/ui-kit.css";
```

Or import it from your JavaScript or TypeScript entry file:

```js
import "piceui/dist/ui-kit.css";
```
